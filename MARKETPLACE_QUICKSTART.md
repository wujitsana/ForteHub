## ğŸ“– User Guide

### Cloning a Workflow (Available Now âœ…)

1. **Browse the Marketplace**
   - Go to https://fortehub.com (automatically redirects to /discover)
   - See all public workflows in beautiful card format
   - Filter by category: Yield, DCA, Rebalancing, Arbitrage, etc.

2. **Find a Workflow**
   - Click "View" to see full details
   - Check description, stats, and creator info
   - Review the source code if available

3. **Clone It**
   - Click "Clone" button on the card
   - Configure any parameters (if applicable)
   - Confirm transaction
   - Workflow deployed to your wallet!

4. **Use Your Clone**
   - Go to Dashboard
   - View your cloned workflows
   - Update parameters as needed
   - Run manually or schedule for automation

### Listing a Workflow (Coming Phase 2 ğŸš€)

**Soon you'll be able to:**

1. **Select a Workflow to Sell**
   - Go to Dashboard
   - Click "List for Sale" on any of your workflows
   - Choose your price in FLOW

2. **Set Your Price**
   - Enter price: 0.1 FLOW, 0.5 FLOW, 1.0 FLOW, etc.
   - ForteHub takes 2% platform fee
   - You get 98% of the sale
   - Example: 1.0 FLOW sale = 0.98 FLOW to you

3. **Confirm Listing**
   - Review listing details
   - Approve transaction
   - Workflow moves to "For Sale" tab
   - Can be bought by other users

### Buying a Workflow (Coming Phase 2 ğŸš€)

**Soon you'll be able to:**

1. **Find a Listed Workflow**
   - Go to "For Sale" tab
   - Browse workflows for sale
   - Click "Buy" to see details and price

2. **Purchase**
   - See price breakdown (seller amount + platform fee)
   - Confirm you have enough FLOW
   - Approve payment transaction
   - Workflow appears in your manager

3. **Use Your Purchase**
   - Treat it like a cloned workflow
   - Same execution and customization options
   - Full ownership and control

## ğŸ’° Understanding Fees

### Clone Fees (Creator Setting)
- Creator can set a **one-time clone fee**
- Default: Free
- Example: "0.5 FLOW per clone"
- You pay when cloning
- Creator gets 95%, platform gets 5%

### Marketplace Fees (P2P Listing - Phase 2)
- **Platform fee**: 2% of sale price
- Example sale: 1.0 FLOW
  - You (seller) receive: 0.98 FLOW
  - Platform receives: 0.02 FLOW
- Fee only charged on P2P sales, not clones

### Total Cost Examples
```
Scenario 1: Clone a Free Workflow
- Workflow Price: Free âœ…
- Gas Fee: Covered by transaction
- You pay: 0 FLOW + gas

Scenario 2: Clone a Paid Workflow
- Workflow Price: 0.5 FLOW
- Gas Fee: ~0.001 FLOW
- You pay: 0.5 FLOW + gas

Scenario 3: Buy from Marketplace (Phase 2)
- Listing Price: 1.0 FLOW
- Platform Fee: 0.02 FLOW (embedded)
- Gas Fee: ~0.001 FLOW
- You pay: 1.0 FLOW + gas
```

## â“ Frequently Asked Questions

### Can I have my workflow listed multiple times?
**No.** Each workflow has one listing. You can unlist and relist with new price.

### Can I change the price after listing?
**Yes!** Click "Edit" on active listing to update price.

### What happens if I unlist my workflow?
**It returns to your manager.** You can list again anytime, or use it privately.

### Can I buy my own workflow?
**No.** The system prevents buying your own listings.

### How do I earn money from listings?
1. Create a great workflow
2. List it for sale
3. Other users purchase it
4. You receive 98% of sale price
5. See earnings in Dashboard (Phase 2)

### Is my workflow copied or transferred?
**Transferred!** When bought, it's moved from seller's manager to buyer's. Seller no longer has it.

### Can I see who bought my workflow?
**Yes!** Sales history shows all transactions (Phase 3).

### What if I want to keep my workflow private?
**Just don't list it!** Clone the workflow yourself, or keep it unlisted in registry.

### How long does a purchase take?
**2-5 minutes** for transaction to seal on Flow blockchain.

### What if I make a mistake in a transaction?
**Cancel it!** Transactions can be canceled before sealing. After sealing, contact support.
